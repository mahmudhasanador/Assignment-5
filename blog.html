Question-1: What are the different ways to select an element in the DOM?

Different ways to select an element in Dom are: getElementById(id) , getElementsByClassName(className) , getElementsByTagName(tagName) , querySelector(selector) , querySelectorAll(selector) etc.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

In JavaScript, "innerHTML" can retrieve the complete HTML content inside an element. But "innerText" can only access the text content of an element and "textContent" can retrieve all text content inside an element including hidden text.

Question-3: What is event delegation in the DOM?

Event delegation is a method that helps us to add one event listener to a parent element without adding separate listeners to each child element.

Question-4: What is event bubbling in the DOM?

Event bubbling occurs when an event starts from the element that was clicked and then moves up its parent elements in the DOM hierarchy.

Question-5: How do you create, add, and remove elements using JavaScript?

We can create element by document.createElement(tagName), we can add element by parentElement.appendChild(newElement) and we can remove element by parentElement.removeChild(childElement).
